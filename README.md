📱 TapAssist

TapAssist is a simple, senior-friendly web app designed to help older adults confidently use everyday technology. With large buttons, clear instructions, and voice guidance, TapAssist makes digital tasks easy and accessible.

🌟 Inspiration
Many senior citizens struggle with basic technology tasks like sending messages, joining online meetings, or identifying scams. TapAssist was created to bridge the digital divide and make technology less intimidating.

💡 What It Does
TapAssist provides:
📲 Step-by-step guides for common tasks
🔊 Voice assistance (reads instructions aloud)
⚠️ Scam awareness tips to keep users safe
👆 Large, easy-to-use buttons and simple interface

🛠️ Built With
HTML
CSS
JavaScript
Web Speech API (for voice guidance)

🚀 How to Use
Open the app
Tap a task (e.g., Send WhatsApp Message)
Follow the step-by-step instructions
Tap 🔊 to hear the instructions read aloud

🎯 Features
Beginner-friendly design
No login required
Works directly in the browser
Fast and lightweight

🧠 Future Improvements
AI-powered assistant for custom questions
